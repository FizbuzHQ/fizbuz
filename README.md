# Fizbuz Web Client

## Basics

- typescript
- react
- react-hook-form
- Apollo Client
- Auth0

## Install

`yarn install`

`cp .env-sample .env`

## Develop

`yarn dev`

## New GraphQL Queries

If you write any new GraphQL queries, you may need to re-generate the bindings in order to execute them:

`yarn codegen`
